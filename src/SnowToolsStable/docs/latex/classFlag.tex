\subsection{Detailed Description}
Stores a rule for a single alignment flag.

Rules for alignment flags can be one of three states:\begin{itemize}
\item NA - All flag values are valid\item Off - \doxyref{Flag}{p.}{classFlag} is valid if OFF\item On - \doxyref{Flag}{p.}{classFlag} is valid if ON \end{itemize}
