\subsection{Detailed Description}
Class to store histogram of numeric values.

The bins of the \doxyref{Histogram}{p.}{classHistogram} are not uniformly spaced, and their ranges determined by partitioning the spans it tablulates into uniform quantiles when initialized by \doxyref{Histogram::initial\-Spans()}{p.}{classHistogram_9e08f9756280ae043ec6bae2bf0f4f1e}. As elements are added and removed this initial bin definition remains constant.
